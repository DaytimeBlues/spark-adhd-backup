# Tech Stack - Spark-ADHD

## Core Stack

- **Languages**: TypeScript, JavaScript
- **Framework**: React Native (0.74.3)
- **Navigation**: React Navigation
- **Animation**: React Native Reanimated
- **Audio**: `expo-av`, `react-native-sound`
- **Web Support**: React Native Web + Webpack

## Data and Persistence

- **Database**: AsyncStorage (Simplicity-first for vibe-coding)

## Infrastructure and Deployment

- **Web**: GitHub Pages
- **Mobile**: Android (Side-loading/Personal SDK)

## External Integrations

- **Google Tasks API**: Task synchronization
- **Plaud API/Webhook**: Voice-to-task automation
